# Simple D boilerplate

Reads all the entries in the provided config.yaml and outputs them.


## Usage

```
dub build
./d-boilerplate -c|--config ${CONFIG_YAML}
```

```
Usage:
-c --config
-h --help      This help information.
```

Sample config.yaml provided in root directory
